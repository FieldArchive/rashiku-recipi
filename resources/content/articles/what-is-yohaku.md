# 余白ってなに？

::me
余白っていう言葉が何度も出てくるけれど、
正直、まだよくわかりません。

余白って、どういうこと？
::

そうですよね。

余白って、
最初は少しつかみにくい言葉です。

広い時間があること、
何も予定がないこと、
ぼーっとすること。

そういうものを想像しやすいけれど、
ここで言いたい余白は、
それだけではありません。

::me
じゃあ、何もしない時間のことではないの？
::

何もしない時間も、
余白を取り戻す助けになることはあります。

でも、
何もしない時間があっても、
頭の中がぎゅうぎゅうで、
ずっと焦っていたら、
余白がある感じはしませんよね。

逆に、
少し忙しくても、
気持ちが通っていて、
自分を見失っていない時は、
余白がある感じがすることがあります。

だから余白は、
単に空いている時間のことではなく、
**自分の中にある、通れる余地**
みたいなものです。

::me
通れる余地？
::

はい。

たとえば、

* 何か嫌なことがあっても、すぐ全部を壊さずにいられる
* イラッとしても、すぐ反応しないでいられる
* 比べて苦しくなっても、自分に戻る道が残っている
* やる気が出ない時にも、自分を全部だめだと決めつけないでいられる

こういう時、
自分の中に少し余地があります。

その余地があると、
気持ちが通ります。

考えも、
呼吸も、
自分らしさも、
全部ぎゅっと詰まらずに済みます。

::me
じゃあ、余白がない時ってどうなってるの？
::

余白がない時は、
少しのことでも全部に響きます。

* ひとつ嫌なことがあると、一日ごと重くなる
* 失礼な態度をされると、ずっと引きずる
* 何もしていないと、自分に価値がない気がしてくる
* やるべきことがあるのに、気持ちが通らなくて止まる

つまり、
出来事そのものより、
**余白がない自分が受けること**
で苦しくなりやすいんです。

::me
余白があると、何が違うの？
::

余白があると、
何も問題が起きなくなるわけではありません。

でも、
問題が起きた時に、
すぐ全部を自分の中心まで入れなくて済みます。

少し待てる。
少しゆるめられる。
少し戻れる。

その `少し` があることが、
とても大きいです。

その `少し` があると、
自分らしさが飛びにくくなります。

::me
じゃあ、余白って増やすものなの？
::

増やすというより、
取り戻す、
守る、
細くなった時にまた通せるようにする、
のほうが近いかもしれません。

余白は、
特別な人だけが持つものではなく、
もともとあるけれど、
焦りや比較や損した感じで見えなくなりやすいものです。

だから、
らしくレシピでやっているのは、
何かすごい自分になることではなく、
余白を取り戻して、
また自分に戻れるようにすることです。

::point
## 余白を思い出すときのレシピ
* 余白は、何もしない時間そのものではない
* 余白は、自分の中にある **通れる余地**
* 余白があると、少し待てる、少し戻れる
* 余白を増やすというより、取り戻す、守る
::
