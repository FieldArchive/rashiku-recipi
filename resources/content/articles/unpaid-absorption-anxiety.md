# 一銭にもならないことにのめり込んでいて、不安な時のレシピ

::me
いまのめり込んでやっていることが、
一銭にもならないことなので、不安になっています。

こんなことに時間を使っていていいのかな、
そんなことをしている場合じゃないんじゃないか、
って思ってしまいます。
::

そういう不安、ありますよね。

しかもそれは、
ただお金が不安というだけではなく、
「成果にならないことをしている自分はだめなんじゃないか」
という感覚まで一緒に連れてきやすいです。

でもここで切り分けたいのは、
**いまお金になっていないこと** と
**価値がないこと** は別だということです。

お金になっていない時間が苦しいのは、
その行為が無価値だからではなく、
「早く意味に変えなきゃ」
「成果にしなきゃ」
という圧が乗って、
気持ちの通り道が細くなっているからかもしれません。

::me
でも現実問題、
お金にならないことばかりしていたらまずいでしょう？
::

その不安はもっともです。

ただ、いま必要なのは、
すぐに価値証明をすることではありません。

不安のまま無理に答えを出そうとすると、
好きだったものまで
「回収しなきゃいけない案件」
に変わってしまいます。

そうなると、
自分を満たすはずの時間まで重くなって、
[余白](/articles/what-is-yohaku)がさらに減っていきます。

::me
じゃあ、どうしたらいいの？
::

まずは、
いまやっていることを
無理に「意味」や「成果」に変えようとしないことです。

この不安が出ている時は、
判断を急ぐより先に、
次のことを切り分けるほうが大事です。

* これは本当に無価値だから苦しいのか
* それとも、お金にならない時間に不安が乗っているだけか
* いま私は、好きなことをしているのか
* それとも、好きなことにまで回収義務を乗せているのか

ここが混ざると、
「好き」も「意味」も両方わからなくなります。

いま必要なのは、
答えを出すことより、
不安で細くなった通りを少し戻すことです。

そのためにできることは、
たとえばこんなことです。

* いまは判断の時間ではない、と決める
* 「一銭にもならない」と頭の中で繰り返すのを止める
* これは将来の価値判断ではなく、いまの不安だと名前をつける
* 好きなことそのものを、すぐ成果化しようとしない

::me
それでも、
「そんなことしてる場合じゃない」って声が出てくるよ
::

はい。

その声は、
現実を見ている声でもあり、
同時に、
不安が強くなった時にすぐ
自分を追い立てる声でもあります。

だから、その声を全部信じなくて大丈夫です。

いまは、
その声に従って走るより、
まず自分の通りを戻すほうが先です。

不安が少し静まってから見ると、
本当に見直すべきことと、
ただ焦っていただけのことが、
前より分けて見えるようになります。

「一銭にもならない」が苦しい時は、
お金の問題だけでなく、
価値の問題まで一緒に揺れていることがあります。

だからいまは、
自分のやっていることをすぐ裁くより、
不安で細くなった通りを少し戻すことを大事にしてください。

::point
## 余白を取り戻すときのレシピ
* **いまお金になっていないこと** と **価値がないこと** を重ねない
* 好きなことに、すぐ回収義務を乗せない
* 「いまは不安が強い」と名前をつける
* 無理に答えを出さず、まず通りを戻す
::
