# 一銭にもならないことにのめり込んでいて、不安な時のレシピ

::me
いまのめり込んでやっていることが、
一銭にもならないことなので、不安になっています。

これをやっていて大丈夫なんだろうかって
落ち着かなくなる。
::

そういうことってありますよね。

つい没頭してしまったけれども、
ふと気づくと不安になる。

たとえば、こんな出来事です。

* 本業そっちのけで趣味に没頭してしまう
* 調べものや学びにずっと吸われる
* 創作や設計ばかりして、売る行動に移れない
* 片づけや生活を整えることばかりしてしまう

気持ちはそっちに向いているのに、
現実感覚のほうが
「それで大丈夫なの？」と止めにくるんですよね。

このテーマの最初の処方は、
「**好きなことをやっている自分を不安で汚さないこと**」
です。

まずは、好きなことをやれている自分を否定しないことです。

その流れ自体は悪いことではないので、
不安を混ぜて濁さないほうがいいんです。

「それで大丈夫なの？」という問いには「大丈夫だよ」って返してあげてください。

::me
大丈夫なの？？
楽しいことをやっていていいの？

全然不安がとれないんだけど。
::


それでは不安解消への解決に向けて、
３つの質問をします。

１つ目
「不安な気持ちになったのはその場のせいだとは思いますか？」

::me
いや、その場のせいじゃないと思う。

だって、やっている最中は夢中だったり、
楽しかったり、
ちゃんと通っている感じがあった。
::

２つ目
「あなたの気持ちはなぜご機嫌な状態からはずれたのですか？」

::me
これって一銭にもならないよねって思った瞬間に、
急に現実が怖くなって、ご機嫌ではいられなくなった。

そう思う前まではやりたいことをやれて、むしろすごい楽しかった。
::

３つ目
「もともとその出来事はずっと不安な出来事でしたか？」

::me
いや、ちがうね。

やっていて、ずっと楽しかった。
楽しすぎて、ふと我に返って、
あれ、これって良くないんじゃない？
って手がとまって、そこから先急に不安にかられて何もできなくなった。
::


解決に向けての答えが出てきましたね。

つまり、
場はもともと悪くなかった、
あなたの機嫌ももともと悪くなかった、
出来事ももともと悪い出来事だったわけではない、
ということです。

::me
え？
どれのせいでもない？
じゃあなんで不安になっているの？

自分がそれをやっていたせいなんじゃないの？
::

あなたの気持ちは、不安にかられる前まではご機嫌な状態だったんですよね。
つまり、それをやっていたこと自体には問題はないのです。

[気分よかったのに、無遠慮に踏みにじられた時のレシピ](/articles/rude-attitude-stuck)で話しましたが、
元々悪かったわけでもないもの達のせいにして、
不安を広げてしまうのでは問題は解決しないのです。

また同じ現象が起きたら何度でも不安にかられてしまいます。

それと同じことが今回もおきています。

今回の出来事に対しても、
現実の何かそのもののせいというより、
「**雑な現象が一瞬起きたせい**」と考えるのが
解決に向けての一番のポイントなのです。

ちょっとわかりにくいですが、
これも、[ざっちー案件](/articles/what-is-zatchy) なんですよ。

::me
ざっちー案件だと思うようにして、
瞬間の雑な想いを飛ばして余白に戻れたとしても、

本来すべきことじゃない、
趣味とか娯楽とか、
楽しいことや楽なことに没頭してしまうことは良くないことなのでは？
::

不安な気持ちを「雑な一瞬の想い」として切り分けられたことはとても良いです。

ただ、
ざっちー案件だと気づいて一瞬の不安からは戻れたとしても、
「でも、そもそもこんな楽しいことに没頭していていいの？」
という疑いが残ると、
またすぐ不安が戻ってきてしまいますよね。

だから今回はもう一段先で、
「本来すべきことではないことに惹かれるのは、本当に悪いことなのか？」
を見直す必要があります。

「楽しいことをやっていていいの？」

その答えをズバリ言うと
「**はい。楽しいことを最優先してやるべきです。**」
となります。

::me
えええ？？

最優先でやるべきなの？
それで本当に大丈夫なの？
::

はい。
最優先で楽しいことをやるべきです。

ただ、ここで一つだけ重要な観点があります。

**本当に心底それが楽しいと思っているか、うきうきするか、やった後にご機嫌になれるか**

です。

余白が生まれる楽しいことと、余白を生まない楽しいことの違いがわかりますか？

一銭にもならない不安がない状態だったらどうかで考えてみてください。

::me
うーーん。
余白を生むのって、気持ちが楽になるってことでしょ。

やったあとでむしろ苦しくなる、
やったことを後悔する、ようなことかな。
::

そうですね。

かなり大事なのはそこです。

やっている最中は楽しくても、
やったあとで苦しくなったり、
自分に戻る感じがなかったり、
後悔や自己嫌悪が強く残るものは、
余白が生まれる楽しいこととはわけて考える必要があります。

つまり、
- 余白に戻る行為

と
- ただ時間や気を持っていかれる行為

とを分けて考える癖をつけてほしいのです。

実践訓練として、以下の事柄がどちらになるか、分けてみてください。

- 本業そっちのけで趣味に没頭してしまう
- 調べものや学びにずっと吸われる
- 創作や設計ばかりして、売る行動に移れない
- 片づけや生活を整えることばかりしてしまう
- SNSやネットニュースをだらだら見てしまう
- 動画を延々見続けてしまう
- ゲームを区切れずに続けてしまう

::me
違いはわからないなぁ。

でも、なんとなく、

- SNSやネットニュースをだらだら見てしまう
- 動画を延々見続けてしまう
- ゲームを区切れずに続けてしまう

この３つはあとで無駄な時間だったなって後悔しそう。
::

質問しておいてなんですが、
じつはこれの答えは人によって違う、が正解です。

例えばＳＮＳやネットニュースをだらだら見たとしても、そういった雑多な意識を取り入れることを生業にしている作家さんとか記者さんとかだと必要なことであり、決して無駄な時間とはいえません。

同じく、
動画の制作をしたいと考えている方、ゲーム制作に関わろうと思っている方などは、すぐに収益にはならないにしても、好きな事であり、時間を忘れて没頭できるくらいのめり込むのは決して悪いことではないのです。

そういう意味では、
「本業そっちのけで趣味に没頭してしまう」だって、趣味がそのうち本業になる可能性だってありうる。
「調べものや学びにずっと吸われる」も無駄な時間ではない。
「創作や設計ばかりして、売る行動に移れない」も、きっかけがあれば花開くことがある。
「片づけや生活を整えることばかりしてしまう」だって、部屋が整うと気持ちいいですよね。

::me
え？
なんなの。

では全部「余白に戻る行為」だというの？
::

そうではないです。

**「あなたが本来やりたいと思っていることに通じているかどうか」**

その強さによって、
同じ事柄でも
「余白に戻る行為」になることもあれば、
「ただ時間や気を持っていかれる行為」になることもあるということです。

::me
なるほど、わかってきたよ。

つまり、私にとってSNSやネットニュースや動画やゲームは
単なる暇つぶしでしかないので、
それを長時間やっても「ただ時間や気を持っていかれる行為」だね。

その他の、
趣味や調べ物に没頭したり、
創作や設計ばかりしたり、
片付けや生活を整える行為は
「余白に戻る行為」だっていうことかな？

::

そうです。

特に、
片付けや生活を整えることは、
余白に戻るのにおすすめな行為です。

よく「物を捨てると頭がスッキリして本来やりたいことが見えてきた」みたいな体験談を聞きませんか？

それだけ、片付けや生活を整えることは、余白を取り戻しやすい行為なんです。

不安に思う気持ちを取り除き、
フラットな気持ちで、本当にやりたいことを最優先にやってみてください。

不安に思う時間がない分、
短時間で好きなことに集中でき、
余白をとりもどしてからやるべき作業に戻る、
というのが一番効率がよいし、
もっというと、
好きなことばかりやっていくほうが、
あなたらしい人生、あなたらしい仕事を見つける近道になるので、
どんどん最優先で好きな事、やりたい事の方へ進む癖をつけていってください。


::point
## 一銭にもならないことにのめり込んでいて、不安な時のレシピ
* 好きなことをやっている自分を肯定する
* 不安になったら、それはざっちー案件だと気づく
* 片付けや生活を整えて余白を生み出す
* 最優先で好きなことをやる癖をつける
::
