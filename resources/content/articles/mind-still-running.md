# 頭の中で考えすぎて、やる気が出ない時のレシピ

::me
やるべきことに対して、やる気が出ません。

何もかも手が出ないくらい重い気分で、
ずっと頭の中で何かが続いていて、
気力がそっちに持っていかれている感じがします。
::

こういう時は、
考え足りないのではなく、
「**もう考えなくていい**」時なのかもしれません。

もう終わったはずなのに、
頭の中ではまだその場面が続いている。

たとえば、

* 納得いかなさ
* 損した感じ
* 気まずさ
* 失礼な態度への引っかかり

こういうものが止まらないまま、
全部の気力を持っていくことがあります。

この時に先にやるのは、
答えを出すことではなく、
**再生を止めること**です。


::me
そうはいっても、
止められないから困っているんだよ
気づくとまたぐるぐる考えてしまっている
::

そうですよね。

止めたほうがいいと分かっていても、
頭の中の再生は勝手に続くことがあります。

だからここでやるのは、
「考えないようにする」ことではありません。

まずやるのは、
「**いままた考えている**」と気づくことです。

何度でも、
「あ、また考えていた」
それだけで大丈夫です。

そこで止めきれなくても大丈夫です。
少し気づけただけでも、
再生の勢いは少しゆるみます。

何度も考えが巡ってしまう時に必要なのは、
「考えないようにしよう」と我慢することではなく、
**頭の詰まりを少しゆるめること**です。

そうやって少し通れるようになった時にできる余地を、
ここでは[**余白**](/articles/what-is-yohaku)と呼びます。

::me
少し詰まりがゆるまった感じがあれば、
そのうち頭の中のループは止まることがある？
::

あります。

ただ、すぐにループが止まらなくても焦らなくて大丈夫です。
気づくたびに少しずつ詰まりをゆるめていくと、
前より長く飲み込まれずに済むようになっていきます。

最初は、
頭の中の考えを止めるというより、
考えにのみ込まれる時間が少し短くなるくらいかもしれません。

それでも十分、通れる余地が戻ってきています。


::point
## 余白を取り戻すときのレシピ
* 考えを止めようとしない
* 「いままた考えている」と気づく
* 頭の詰まりを少しゆるめる
::
