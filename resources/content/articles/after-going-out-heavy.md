# 人に会ったり、出かけたりしたあとから重い時のレシピ

::me
人に会ったり、出かけたりしたあとから、
なぜかずっと気分が重い。

別に嫌なことがあったわけでもないし、
すごく落ちこんでいるわけでもないのだけど。
::

それは、
あなたの中にまだ消化できていない何かがあって、
それがまだ内側で動き続けているんですね。

いまのあなたは[余白](/articles/what-is-yohaku)がある状態ですか？

::me
うん、
余白あるはずなのに、
内側だけまだ終わっていない感じがするんだよね。

まるで船酔いみたいに、
内側がぐるぐるしてて、気持ちが悪い。
::

なるほど。
[ざっちー](/articles/what-is-zatchy)のせいではなさそうですか？
人や場や自分の雑な状況に反応した可能性はないですか？

::me
うん、
嫌な想いをさせられたわけではない。
嫌な場だったわけでもない。
卑屈になってしぼんだわけでもない。

ただ、慣れてない状況ではあった。
人としっかり対話をして、
それが心に引っかかっていて、
まだ消化しきれていない状況かなって思う。

でもこんなに長い間、
無意味に時間をとられちゃうのもったいないんだよね。
::

余白で満たされた状態になれたら、
こんな時間、無意味だと思ってしまいますよね。

もっと違う自分らしいことに時間を使いたいと。

しかも、ざっちーのせいでもないのだから、
余白を整えて解決しようがない。

実は**平和になったからこそ、**
**誤魔化せなくなったものが見えている状態**なのです。

::me
ざっちー案件でなくて、
余白も満たされているのに、
それだからこそ、誤魔化せなくなったものがあるってこと？

そんな奥底に、まだ整理されていないものがあったの？
::

余白を満たすことでほとんどの雑事、ざっちー案件は解決しますが、
それだけでは今回のようなモヤモヤははずれません。

**もう安全だから、蓋をしていた願いが浮いてきた**と思っていいです。

整った状態だからこそ、
* 見失っていたもの
* 蓋をしていたもの
* あきらめていた世界

を、まっすぐ見せつけられてしまった。

モヤモヤの正体は、
未整理の不安というより、
**まだ自分の中に生きている願いに、**
**正面から触れてしまった衝撃**なのです。

つまり今あなたに必要なのは、
それを早く忘れることではなく、
**ああ、私はまだこれを望んでいたんだと認めること**です。


::me
ああ、
確かに昨日は
蓋をしていたあきらめていた世界を
見せつけられた日だった。

それを認めたら解放される？
うーん、されてないかも。

どうしたらスッキリできる？
::

認めただけではまだ解放されないという場合、
紙に書き出してみることをお勧めします。

**モヤモヤの正体をきちんと見破ること**です。

私は何を失ったと思っているのか
ではなく、
**私は何を欲しいと再確認させられたのか**
を見たほうが進みます。

ただ漠然と雑にひとまとめで
あの世界を望んでいたんだ
としている限り、あなたの心は重いままです。

見えたものに置き場を与えることで、
昇華される可能性が高まります。

そうか、
私はこういう世界に惹かれたんだ
私はこういう場に入りたいと思ったんだ
とわかるくらい、しっかり深掘りしてください。

::me
昨日見せつけられたのは、
余白があって、
自分を持っていて、
人同士が優しくつながっている
楽しそうな世界だった。

そういう空気の中にいたい。
そういう人たちの中で、
私のまま関わっていたい。
::

そうです。
そこまで書いてみて、
いまどう感じていますか？

::me
いまの生活に決して
不満があるわけではなかったけれど、
そういう願いが心の中にあったんだなぁって
再認識させられたよ。

で、確かにちょっとモヤモヤが解消された。
::


余白がある状態で、
自分の中にとどまっているモヤモヤの正体をきちんと見破ることができると、
次のステージに進むことができます。

だから、
人に会ったり出かけたりしたあとに
気分が重くなったら、
「チャンス！」と思ってください。

たとえば、
いまの見破りのように、
そういう空気の中にいたいと思えた時、
そういう世界に行きたいと思っている自分も、
まだそこに行けていない自分も、
同時に許してあげられることがあります。

あの人たちの世界が、
本当に自分の想像した通りかどうかはわからない。
でも、
昨日たしかに私は、
そういう空気に惹かれたんだ、
と認められた時、
自分の中の揺れは少しおさまります。

一度では無理かもしれません。
場に行くたびに、ぞわぞわすることが続くかもしれません。

でもそうやって、
余白を保ったままで付き合い続けていたら、
仲間として入れる日が来るかもしれないし、
そういう仲間に出会っていく兆しも、徐々に見えてきます。

出会いを求める気持ち、
そういう場所がほしいと思う気持ちは、
消さずにふくらませておいてください。

自分の本当に言いたいことを、
座談会で平気で言えるような場所があったらいい。
もしそんな思いが心に灯されたのなら、
大事にしておいてください。

ある程度その気持ちが高まってきた時、
それに応えるような兆しは、
かならずあらわれます。

そう思って重い気持ちを楽しみに変えていってみてください。



::point
* まだ消化できていない何かが、内側で動き続けていると気づく
* ざっちーのせいでも余白がないせいでもない時にこそ起きると気づく
* モヤモヤの正体を書き出してきちんと見破る
* いまの重い気持ちをチャンスととらえて楽しみに変える
::
