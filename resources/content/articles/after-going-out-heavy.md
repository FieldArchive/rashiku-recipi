# 人に会ったり、出かけたりしたあとでモヤモヤが晴れない時のレシピ

::me
人に会ったり、出かけたりしたあとから、
なぜかずっとモヤモヤとそのことを考え続けています。

別に嫌なことがあったわけでもないし、
すごく落ちこんでいるわけでもないのだけど。
::

それは、
あなたの中にまだ消化できていない何かがあって、
それがまだ内側で動き続けているんですね。

いまのあなたは[余白](/articles/what-is-yohaku)がある状態ですか？

::me
うん、
余白あるはずなのに、
内側だけまだ終わっていない感じがするんだよね。

まるで船酔いみたいに、
内側がぐるぐるしてて、気持ちが悪い。
::

なるほど。
[ざっちー](/articles/what-is-zatchy)のせいではなさそうですか？
人や場や自分の雑な状況に反応した可能性はないですか？

::me
うん、
嫌な想いをさせられたわけではない。
嫌な場だったわけでもない。
卑屈になってしぼんだわけでもない。

ただ、慣れてない状況ではあった。

人との予期せぬ交流が心に引っかかっていて、 
まだ消化しきれていない状況かなって思う。
::

前提を崩されたことが
モヤモヤの入口になることがあります。

こうはならないだろうという想いを覆されたショックが、
モヤモヤのきっかけになっていませんか？

良いと言われると思っていたのに、否定された。
まさか声をかけられないだろうと思っていた場面で声をかけられた。
平穏な日常を何かが不意に越えてきた。

これはざっちー案件ではなく、
事実としての予期せぬ出来事です。

余白で満たされていても起きる現象です。

こういった出来事が、
なんでこんな予期しないことになったんだ？
とモヤモヤ考え続ける原因になりがちです。

::me
たしかに前提を崩された出来事があった。

でもこんなに長い間、
モヤモヤ考え続けるの嫌だな。

無意味に時間をとられちゃうのもったいないんだよね。
::

余白で満たされた状態になれたら、
こんな時間、無意味だと思ってしまいますよね。

もっと違う自分らしいことに時間を使いたいと。

しかも、ざっちーのせいでもないのだから、
余白を整えて解決しようがない。

嫌なことがあったわけでも
落ちこんでいるわけでもないのに内側から消えない場合、
実は**平和になったからこそ、**
**誤魔化せなくなったものが見えている状態**なのです。

::me
ざっちー案件でなくて、
余白も満たされているのに、
それだからこそ、誤魔化せなくなったものがあるってこと？

そんな奥底に、まだ整理されていないものがあったの？
::

余白を満たすことでほとんどの雑事、ざっちー案件は解決しますが、
それだけでは今回のようなモヤモヤははずれません。

いまのあなたの状態は、
**いままで蓋をして忘れていた自分らしさが浮いてきた**
と思っていいです。

整った状態だからこそ、
* 見失っていたもの
* 蓋をしていたもの
* あきらめていた世界

にすっとつながった。

モヤモヤの正体は、
未整理の不安というより、
**まだ自分の中に閉じ込められていた自分らしさに、**
**正面から触れてしまった衝撃**なのです。

つまり今あなたに必要なのは、
それを早く忘れることではなく、
**ああ、私はそんな風になりたいんだと認めること**です。

認めるとほどけていきます。


::me
蓋をしていた自分らしさ？？

なにに蓋をしていたか、
なにを望んでいるのか、
まだよくわかってないんだけど。
::

その人と交流した時に、
自分もこの人みたいにありたいという感じとか、
こんなふうに人とつながれたらいいのにとか、
こんな場面で
自分は本当はこんな風に接したかった
けれどできてなかったなぁ

みたいな気持ちが
ふっと浮かんできませんでしたか？

それが、
あなたがまだほどいていなかった、
蓋をしていた本来の自分らしさです。


::me
それを認めたら解放される？
うーん、されてないかも。

どうしたらスッキリできる？
::

認めただけではまだ解放されないという場合、
紙に書き出してみることをお勧めします。

**モヤモヤの正体をきちんと見破ること**です。

私はなぜこのことを見失っていたのか
ではなく、
**私はどうなりたかったと再確認させられたのか**
を見たほうが進みます。

自分を見失ったトラウマの原因を探るのではなく、
本来なりたかった自分をまっすぐ見つめ直す。

見えたものを素直に文字化することで、
昇華される可能性が高まります。

そうか、
私はこういう世界に惹かれたんだ
こういう場に入りたいと思ったんだ
こういう自分になりたかったんだ
とわかるくらい、しっかり深掘りしてください。

そこまで書いてみて、
いまどう感じていますか？

::me
いまの生活に決して
不満があるわけではなかったけれど、
そういう願いが心の中にあったんだなぁって
再認識させられたよ。

で、確かにちょっとモヤモヤが解消された。
::

余白がある状態で、
自分の中にとどまっているモヤモヤの正体をきちんと見破ることができると、
次のステージに進むことができます。

だから、
人に会ったり出かけたりしたあとに
いつもと違う感覚が残ったら、
「チャンス！」と思ってください。

そのつど文字化して、
いまの自分と本来の自分らしさとの答え合わせをしていくのです。

そうすることで徐々に
また同じシチュエーションが起きた場合、
「あ、これはあの時と同じだな」
と気づけるようになって、モヤモヤが薄くなっていきます。

自分の中の揺れは段々とおさまっていきます。

一度では無理かもしれません。
予期せぬ出来事が起こるたび、ぞわぞわすることが続くかもしれません。

でもそうやって、
余白を保ったままで居続けていたら、
予期せぬ出来事でも余白は飛ばないし、
自分を保ったまま、
こうありたいという自分のままで
人と接することができるようになっていきます。

そう思って内側で引っかかっている気持ちを
楽しみに変えていってみてください。


::point
* まだ消化できていない何かが、内側で動き続けていると気づく
* ざっちーのせいでも余白がないせいでもない時にこそ起きると気づく
* モヤモヤの正体を書き出してきちんと見破る
* いまの引っかかっている気持ちをチャンスととらえて楽しみに変える
::
