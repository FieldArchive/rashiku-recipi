# 失礼な態度そのものが引っかかっている時のレシピ

::me
失礼な態度を取られたこと自体が、
ずっと引っかかっています。

あの言い方、あの空気、
ああいう返し方をされたことが、
あとから何度もよみがえってきます。
::

それは引っかかりますよね。

しかも苦しいのは、
ただ嫌なことをされたからだけではなく、
「なんであんな態度を取られなきゃいけなかったんだろう」
という納得いかなさが残ることだと思います。

たとえば、
普通に聞いただけなのに雑に返されたり、
こちらは落ち着いていたのに、
相手だけが妙にトゲのある態度だったり。

失礼な店員とのやりとりみたいに、
その場ではなんとか終わったのに、
あとからじわじわ引っかかることがあります。

::me
そう。
その場では終わったんだけど、
あとからずっと引っかかるんだよ。
::

こういう時にまず大事なのは、
その引っかかりを
「たいしたことじゃない」と消さないことです。

失礼だったものを、
失礼だったと認める。

そこを曖昧にすると、
自分の感覚まで鈍くなります。

::me
でも、気にしすぎなのかなとも思ってしまう。
::

そう思ってしまいますよね。

でも、
気にしすぎかどうかを先に裁かなくて大丈夫です。

先に見たいのは、
その失礼さを
自分の中まで入れてしまっていないかどうかです。

失礼な態度を受けると、

* なんであんな言い方をされたんだろう
* 私が変だったのかな
* 言い返したほうがよかったかな
* あの場で何かできたかな

と、何度も中で再生しやすいです。

でもそのたびに、
相手の雑さをもう一度自分の中に通してしまうことにもなります。

::me
じゃあ、どうしたらいいの？
::

まずは、
その場をもう終わった出来事として扱うことです。

相手を理解しようとしすぎない。
正そうとしない。
意味を探しすぎない。

「あの人は雑だった」
それだけで止めてよいこともあります。

そして、
その空気から一度離れることです。

* 場所を変える
* 飲み物を飲む
* 外の空気を吸う
* 別のことに視線を移す

こういう小さいことで、
相手の雑さが自分の中に残り続けるのを少し止めやすくなります。

::me
でも、あとからまた思い出しちゃうよ。
::

思い出しますよね。

その時は、
解決しようとしなくて大丈夫です。

必要なのは、
相手を理解することでも、
その場の正解を見つけることでもなく、
これ以上その雑さを自分の中で再生しないことです。

失礼な態度そのものが引っかかっている時は、
相手を変えるより先に、
自分の中に入れすぎないことが大事です。

::point
## 余白を取り戻すときのレシピ
* 失礼だったことを、失礼だったと認める
* 気にしすぎかどうかを先に裁かない
* 相手の雑さを、自分の中で何度も再生しない
* その場を終わったこととして、少し距離を取る
::
