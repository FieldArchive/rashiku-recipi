# 気分よかったのに、無遠慮に踏みにじられた時のレシピ

::me
気分よくしていたのに、
急に無遠慮に踏みにじられて、
それがずっと引っかかってしまっています。
::

それは引っかかりますよね。

しかも苦しいのは、
ただ嫌なことをされたからだけではなく、
せっかく通れていた流れを
急に向こうの雑さで切られた感じがすることだと思います。

たとえば、
こちらは普通に気分よくいたのに、
雑に接し返してきたレストランの店員、
急に断りもなく肌に触ってきたヨガの講師、
無遠慮に言われたくないことを言ってきた知人など。

そういう時は、
予防線を張る間もなく向こうの場に引っ張られて
自分の気持ちが乱されてしまいがちです。

::me
そう。
私はご機嫌でいられたはずなのに、そこを崩されて、
しばらくは元のご機嫌な状態に戻れなくなってしまうの。

そうなると、せっかくの美味しい料理やレストランも悪い印象になるし、
楽しくリラックスできるはずのヨガの時間もひどいものになる。

知人とのやりとりでも、
本当は普通に過ごせたはずの時間や関係まで、
その一言のせいで思い出したくもない嫌な感じになる。
::

なによりつらいのは、
本当はよかったはずの場まで
一瞬の雑な出来事のせいで悪く染まってしまうことだと思います。

料理が悪かったわけじゃない。
ヨガそのものが悪かったわけでもない。
知人との関係そのものが元々悪かったわけでもない。

でも、
その人の一瞬の無遠慮さのせいで、
その場やその人全体が嫌な記憶みたいになってしまう。

関係や時間全体が汚染されたように感じられるわけです。


::me
本来、料理美味しかったな、ヨガ気持ちよかったな、
会話楽しかったなで終わってたはずなのよ。
でもそんなことをされたせいですべてが台無しになったのよ。

これ、ほんとなんとかしたい。
::

解決に向けて考えてみましょう。

３つ質問をします。

１つ目
「嫌な気持ちになったのはその場のせいだとは思いますか？」

::me
もちろん、場のせいじゃないよ。
その出来事が起きる瞬間までご機嫌な状態だったもの。

場が悪くなかったから逃げずに直撃食らっちゃったわけで、
むしろ場の雰囲気があきらかに悪かったら、
心を準備ができて、ここまで落ち込まずにいられたと思うよ。
::

２つ目
「あなたの気持ちはなぜご機嫌な状態からはずれたのですか？」

::me
ほんとに思いもしない瞬間に食らったからだね。
完全に無防備だった。

常に[余白の体幹](/articles/what-is-yohaku)を感じてご機嫌な状態をキープしようと思っていても、
無防備になるときはどうしてもできちゃう。
ほんとにご機嫌な時こそ無防備になるのかも。
::

３つ目
「もともとその人はずっと無遠慮な人でしたか？」

::me
いや、急な無遠慮攻撃だった。

たぶんだけど、向こうは悪気があってやったわけじゃなかったんだと思う。
私にとっては絶対やってほしくなかったことだけど、
それは向こうは気づいてなかった可能性があるね。

だから直撃食らったのかもしれない。

レストランの店員もヨガの講師も知人もみんな、
最初っから変な人だったらこちらも警戒できてたので、
ここまで深くこちらが傷つくことなかったかもって思う。
::

解決に向けての答えが出てきましたね。

つまり、
場はもともと悪くなかった、
あなたの機嫌ももともと悪くなかった、
相手ももともと悪い人だったわけではない、
ということです。

::me
え？？

じゃあ何も悪くないってことじゃない。
解決になってない。

なんで私は台無しにされたの？
::

今回の出来事に対しては
「**雑な現象が一瞬起きたせい**」と考えるのが解決に向けての一番のポイントです。

場のせいでも、あなたのせいでもなく、
今回のように相手に悪気がないだろうとわかる件では、
相手本人のせいにしすぎないことです。

そのどれかに強く紐づけた瞬間に、
あなたはバッドエンドルートに突入してしまいます。

::me
そんな風にいわれても、
嫌なことを言われたりされたりした瞬間に
このモヤモヤをどこにぶつけたらいいかわからないよ

誰も何も悪くないなんて。
そんな失礼なことする相手がどう考えたって悪いじゃない。

そんな風に考えてしまう自分だって未熟だと思うし。
::

そこです。
それがバッドエンドの入り口です。

まず自分が未熟だと思うのはやめましょう。
失礼なことをされて、そんな風に考えるのは当然なことです。
ただ、そう考える必要はないし、考えれば考えただけ嫌な気持ちが増すだけです。

それよりも、失礼なことをされても気にしないルートを選んだ方がいいという話です。


::me
わかった。
バッドエンドはもう選びたくない。

でも失礼なことをされたら気にはなるよ。
相手が悪いでしょ
::

相手が悪いと思っているうちは、まだバッドエンドルート上です。

あなたも３番目の問いで答えていますが、相手は最初からずっと無遠慮だったわけでも、わざと失礼な態度をとってきたわけでもないですよね。

むしろいきなり失礼になったから深く傷ついたし、そんな風に悪くとらえる自分が悪いのでは？とまで行きついてしまった。

だから相手が悪いわけでもない。

となると誰も何も悪くなくて「**その瞬間だけが悪かった**」ということになります。

::me
その瞬間だけが悪かった？
::

はい。

「本当はよかったはずの場が一瞬の雑な出来事のせいで悪く染まってしまうこと」

これはわりとよくある現象です。

これをいちいち、
その場のせい、失礼な相手のせい、そんな風に考える自分のせい
にしていたらキリがありません。

元々悪かったわけでもないもの達のせいにするのでは解決しないのです。
同じ現象が起きたら何度でも同じ残念な思いにかられてしまいます。

そのエンドではないルートに向かう、魔法の呪文があります。

「ああ、これは **[ざっちー案件](/articles/what-is-zatchy)** だな」
です。

::me
ざっちー案件？
なにそれ。
::

「ざっちー」という名の「雑な何か」が相手に憑依した、そのせいで、その瞬間に相手は私にそんな対応をしてきたのだ、ととらえてしまうほうが、余白ルートに近づけます。

その人が一瞬の「雑な何か」にとりつかれたんだなって試しに思って見てください。

そうすると、どう感じますか？

::me
うーん。
少なくとも、
自分の不愉快の原因を、
その人本人のせいにはしないかも。
::

そう、それが大きな前進です。

もし明らかに故意に本人が嫌がらせであなたに対してやっていたとしたら、はっきりと逃げるべきです。

ただ、今回の件はどれもそうではないですよね。
悪気がないだろう相手から、急に無遠慮に踏みにじられて、場を台無しにされている。


その人自身は普段からはちゃめちゃなわけではない場合、
そんな場合は、本来、本当にその人本人のせいではないのですから。

そうなると、問題は人のせいではなく、「ざっちー」という現象だと思ってしまうほうが物事の本質に近い。

だから、こういう現象がおきたときは、「ああこれはざっちー案件だな、きたなこいつ」って思うようにすると劇的に機嫌の快方に向かいやすくなるのです。

場のせいではない、自分のせいではない、でも相手のせいじゃない？って思いが抜けない時に効果的です。
ぜひ、思い出して試してみてください。


::point
## 無遠慮に気分を踏みにじられたときのレシピ
* 自分のせいではないと気づく
* 場のせいでもないと気づく
* 悪気がなくやられた場合、相手のせいでもないと気づく
* ざっちー案件だな、と気づく
::
