# 雑に時間を使ってしまっている時のレシピ

::me
気づくと
ネットニュース見続けて
時間がとけてる。

別にストレスからとか
何かから逃げたいからとかではなく、
見た後にすごく後悔するとかでもないんだけど、
雑に時間を使っちゃっててなんだかなって思う。
::

それだけ時間を雑に使っていても
後悔してネガティブな落ち方をしていないのは、
そこまで大きく[余白](/articles/what-is-yohaku)を失っていない状態とも言えます。

それ自体は悪いことではないのです。

ただ、
何かを強くやりたいわけでもないまま、
何もすることがないし、
ひとまず時間を潰すか、
くらいの感覚で、
雑に自分の時間を溶かしている状態ですね。

::me
そこまで大きく余白を失っていないから？
ってことは少しは余白を失っているってこと？
::


正確にいうと、
まだ余白が満たされた状態を
うまく自分で扱えていないということです。

だから、
せっかく余白があっても、
その時間をどう使ったらいいのか
わからなくなっているんですね。

今までは、
[ざっちー](/articles/what-is-zatchy) に急かされたり、
損得で動かされたりしていたぶん、
何もしなくていい時間の中で
自分がどう過ごしたいかを
あまり上手に育ててこられなかったのでしょう。

その結果、
苦しくはないけれど、
少し手持無沙汰で、
なんとなくスマホで時間を溶かしてしまう。

これは悪化ではなく、
余白のある時間の使い方に
まだ慣れていないだけです。


::me
え。それは嫌だな。

せっかく余白の体幹を整えて
わりと幸せのなかにいるのに。

でも、たしかに、
なにをしたらいいかわからなくなってる。
::

なにをしたらいいかわからない場合は、
まず、**なにもしない状態に慣れる** ことをおすすめします。

たぶん、
いままでぎゅうぎゅうに予定をつめることこそ人生だと思っていた場合は
それは苦痛な時間に感じられるかもしれません。

でも、
余白で満たされた状態そのままでいること、
自分の外側にある世の中のことではなく、
自分のことだけを感じることに、
少しずつ慣れてほしいのです。

これは、
ある意味では禅のようなものです。

余計なことを足さず、
いまの自分の状態のままでいることに、
少しずつ慣れていくのです。


::me
えーー

むずかしい！
何か考えたりしちゃう。
::

そうですね。
難しいと思います。

でも、
何か考えてしまっても大丈夫です。

何も考えないことを目指すというより、
考えていたことに気づいたら、
また今に戻るのです。

自分の呼吸の音、
周りから聞こえる音、
身体の中、
吸って吐いてをして動いている感覚。

こういった
いまの状態を
素直に感じるのです。

それができるようになると、
余白のある時間の中でも
自分らしくいられるようになっていきます。

::me
ネットニュースを見る時間が
禅を体得する時間か。

なんかかっこいいな。
::

そうですね。
かなりかっこいいです。

しかも、
無理に立派なことをするというより、

雑に時間を溶かしたくなった時に、
少しだけ自分に戻る
という感じなので、
かなり実用品です。

ネットニュースを見る代わりに
いきなり大きなことをしなくてよくて、

* 呼吸を見る
* 音を聞く
* 身体の感じをみる
* そのまま少し座る
だけでもう、
時間の使い方が全然違ってきますよね。

それができたら、
ほんとうに
**余白の体幹で生きてる人**
っぽいです。

あと他にも
空いた時間にできるおすすめな行動があります。

::me
なに？
知りたい
::


それは、
**時間を忘れるほど没頭して、ついやってしまうことをやること**です。

急に時間があいても
思い出せないと思うので
普段から紙にかいておくことをお勧めします。

やらなくてはいけないことがあるときにかぎって
ついやってしまいたくなることがあるはずです。
それを覚えておいてください。

詳しくは
[別のことに没頭してしまう時のレシピ](/articles/avoidance-escape) に書いてあります。

ぜひ、
せっかく培った土台をうまく育てて、
余白に満たされた状態で楽しくやりたいことに没頭してみてください。




::point
* 雑な時間の使い方も、それ自体は悪いことではないと気づく
* なにもしない状態に慣れる
* 禅のような在り方を意識して、余白の体幹で生きてる人になる
* 時間を忘れるほど没頭して、ついやってしまうことをやる
::
