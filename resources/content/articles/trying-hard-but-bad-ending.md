# なぜかバッドエンドになってしまう時のレシピ

::me
対応を気をつけていたはずなのに、
なぜかバッドエンドになってしまうことがある。

失礼な態度を取られたり、
場の空気が悪くなったり、
あとからずっと引っかかる終わり方になることがあります。
::

それはしんどいですよね。

しかも苦しいのは、
ただ嫌な目にあったからだけではなく、
「対応を気をつけていたのにこうなった」
という感覚が残ることだと思います。

だから余計に、
自分が悪かったのか、
相手がひどかったのか、
何を間違えたのか、
頭の中で何度も見直したくなります。


::me
不思議なのがさ、
うまくいくときもあるのよ。
必ず不穏で終わるわけではなく、良好な感じで終わることもあるの。

その差はなんだろう？
何が違うんだろう？
::

そのことに気づけたのは大きいです。

**相手ではなく自分にあるのかもしれないな**と考えられた時、さらに真理に大きく一歩近づきます。

同じように失礼なことを言われても、
関係がこじれなかった時があった。場が荒れなかった時があった。自分の心象が違った時があった。

それらの時のことを思い出してみてください。

その時は、
相手の雑さを目の当たりにしても、
なぜかそこまで気にならずにいられませんでしたか？

その差が、
バッドエンドになるかどうかの差なのです。

つまり、相手そのものより、
**その時の自分に[余白](/articles/what-is-yohaku)があるかどうか** がその後のエンドに影響を及ぼすのです。

余白があると、相手の雑さにそのまま巻き込まれずに済むことがあります。

そこまで気づけたら、あとは余白を意識できる瞬間が増えていけば、バッドエンドを選ばずにすむようになれる日がきます。

::me
どうやったら余白がある状態になれるのでしょうか？
その場その時に余白が意識できない場合は、詰みってことですよね。
::

詰みではないです。

ただ、その場で急に余白を作ろうとしても、間に合わないことがあります。

だから大事なのは、
嫌なことが起きてから慌てて余白を探すことではなく、
**ふだんから余白がある感じを知っておくこと**です。

自分の中の余白のあるなしを感じとる訓練をしておくことで、非常時に気づけるようになっていきます。

::me
ふだんから？
ふだんってどんな時のことをいうの？
::

日々の一瞬一瞬のことです。

特に、
何気ない時、落ち着いている時、幸せを感じる時にこそ、
「いま余白があるな」
「いまは通っているな」
と感じる練習をしておくことをおすすめします。


余白がない時とある時を、
こまめに感じ分けるようにしておくんです。

そうすると、
非常時に
「あ、いま詰むやつに入りかけてる」
と気づけるようになります。

::me
その場で気づけたら、何が変わるの？
::

大きく変わります。

一瞬でも余白が意識できたら、バッドエンドになる可能性がかなり低くなります。

気づけるだけで、
相手に影響されるまま反応して、
バッドエンドの言葉や態度を返してしまう前に、
止まれることがあるからです。

たとえば、
「すみません、ちょっと頭が急にいっぱいになったので、少し空気吸ってきます」
と離れることもできるようになります。

その一手があるだけで、
エンドはかなり変わります。



::point
## バッドエンドにならないためのレシピ
* ふだんの良い状態の時から余白のあるなしを意識しておく
* 相手の態度に強く反応しそうになったら、余白がない合図
* 余白がないと感じたら、一度その場から距離をおく
::
