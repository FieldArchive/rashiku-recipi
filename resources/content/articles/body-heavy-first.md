# 原因不明で身体が重い時のレシピ

::me
やるべきだと思うんだけど、どうにもやる気が出ません。

何か出来事があったせいというよりも、
身体が重くて、何も手につきません。
::

それはしんどいですね。
やるべきことは見えているのに、身体が重くて手につかない時ってあります。

そういう時は、「怠けている」というより、
まず身体の通りが悪くなっていることがあります。

今必要なのは、
前進ではなく、
気持ちの通り道を少し広げることです。

「身体の通りを良くする効果的な方法」としては次のようなものがあります。

* **水分をとる（白湯がおすすめ）**
* **ソファやベッドがあれば、横になって目をつぶってみる**
* **伸びをする、深呼吸してみる**
* **全身の力を抜いて口角をあげてみる**

身体が重い日は、やる気を出そうとするより、少し身体をほぐしたり、疲れを癒したりすることで、詰まりが取れて動けるようになることが多いです。

「ちゃんとやろう」ではなく、
詰まりを1ミリゆるめるで十分です。


::me
「身体の通りを良くする方法」で普段なら軽くなれるんだけど、
それでもつらいときはどうしたらいいの？
::

本当に重い日は、
何かを一つやることすら負荷になります。

そういう日は、
やることを決めなくて大丈夫です。

水分をとる。
横になる。
目を閉じる。

それすら無理なら、今はこれだけで十分です。
**「これで大丈夫、これでいいんだ」**

::me
なにもやらないでいると自分がダメなやつだと思えてきて、
それがつらいんです。
::

ここで切り分けたいのは、
**何もしていないこと** と
**自分の価値** を重ねないことです。

止まっている時間に、
だめ判定まで流れ込んでくると、
休めないうえに、ますます動けなくなります。


身体が重い時は、

* やる気を無理に出そうとする
* 予定を見直す
* 自分を責める
* 正しい過ごし方を探しにいく

こういったことをしない方が戻りやすいです。


::me
やるべきことはあるんです。
こんなふうに止まっていていいのかなって思います。
::

その気持ちはもっともです。
「止まりたい」わけではなく、止まっている場合じゃないのに止まってしまっている感じが苦しいんですよね。

いま必要なのは、**無理に進むことではなく、戻れる形をつくること**です。

いまのあなたは、さぼろうとしているのではなく、
動こうとしても通りが悪くて止まっている。
その状態で「こんなふうじゃだめだ」と押すと、前に進むどころか、ますます固まります。

だから、「止まっていていいのかな」に対する答えは、
このまま固まるのはつらいけれど、いまは無理に走り出す場面でもない です。

戻れる形をつくること を目標にするとよいです。


::me
戻れる形をつくること？
それが、気持ちの通り道を少し広げるっていうこと？
::

はい。

「成果を出す」ではなく、
「通りを戻す」です。

身体が重い時に必要なのは、
意志の強さで押すことではなく、
詰まりをゆるめて少し**通りを戻すこと**です。
そうやって少し通れるようになった時にできる余地を、ここでは[**余白**](/articles/what-is-yohaku)と呼びます。

だからいまは、
何かを進めることより、
この通りを少しでも戻せるほうを大事にしてください。

::point
## 余白を取り戻すときのレシピ
* 「これで大丈夫、これでいいんだ」と心から思う
* 「何もしていないこと」と「自分の価値」を重ねない
* 無理に進まず、詰まりをゆるめる
* 少しでも気持ちが通るほうを選ぶ
::
