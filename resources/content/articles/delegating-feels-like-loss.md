# 自分でできることにお金を払うのは損だと思ってしまう時のレシピ

::me
頼めば楽になるのはわかっているのに、
「これくらい自分でできる」
「こんなことにお金を払うのは損」
と思って、結局なんでも自分でやってしまいます。

自分でできなくはないから、
なおさら任せにくいです。
::

それは、ただの節約癖というより、
**自分でできることにお金を払うのは負け**
みたいな感覚が混ざっているのかもしれません。

でもここで苦しくなりやすいのは、
「自分でできるか」
だけで判断してしまうことです。

::me
だって、
自分でやれば無料みたいなものじゃない？

お金を払わずに済むなら、
そのほうが得な気がする。
::

一見そう見えますよね。

でも本当は、
自分でやることは無料ではありません。

そこには、
時間、
気力、
判断力、
[余白](/articles/what-is-yohaku)、
家族や自分に向けるやわらかさまで使われています。

だから判断軸は、
**自分でできるかどうか**
ではなく、
**これは自分の命や時間を使ってまで、自分がやることか**
のほうが近いんです。

::me
なるほど。

たしかに、
できることと、
自分がやるべきことって、
同じとは限らないね。
::

そうなんです。

自分でできる。
でも自分がやると削れる。
自分がやると他の大事なことが止まる。
自分がやらないことで全体が進む。

そういうものは、
任せる価値があります。

任せることはぜいたくではなく、
**自分の核を守るための配分**
なんです。

::me
でも、
器用な人ほど自分でできちゃうから、
抱え込みやすいよね。

それなりにできてしまうから、
困っていることに気づきにくい。
::

まさにそこです。

自分でやって満足がいくと、
その運用は壊れません。

だからこそ、
「できたかどうか」だけでなく、
「これを自分で抱えることで、世界が細くなっていないか」
を見る必要があります。

頼まない。
払わない。
なんでも自分でやる。

それで一時的に気持ちよく回っても、
気づかないうちに、
余白や選択肢や人を通してひらく道まで細くしていることがあります。

::me
じゃあ、
どう見直せばいいの？
::

まずは、
「自分でできるからやる」
をそのまま採用しないことです。

その代わりに、

これは自分の核に関わることか。
自分がやることで力が戻ることか。
それとも、
自分でもできるけれど、
やると不機嫌になったり削れたりすることか。

そこを分けてみてください。

後者なら、
任せることは損ではありません。
むしろ、
**お金より余白を守ったほうが得**
な場面かもしれません。

::point
## 自分でできることにお金を払えない時のレシピ
* 「自分でできる」と「自分がやるべき」は同じではない
* 自分でやることは無料ではなく、時間や気力や余白を使っている
* 任せることはぜいたくではなく、自分の核を守るための配分
* 器用でできてしまう人ほど、抱え込みが運用として固定しやすい
* お金を守るだけでなく、自分の命や時間を何に使うかで見る
::
