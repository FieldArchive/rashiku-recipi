# 大事な場で、自分らしさが飛んでしまう時のレシピ

::me
大勢の初めての人たちがいる場に行くと、
変に気負ってしまい、
自分らしさが飛びそうになる。

本当は人が好きだし、
ちゃんとその場を感じたいのに、
「私はこういうの苦手かも」
「ここは違うかも」
って、自分から閉じてしまうことがあって、
もうそれをやめたいです。
::


それは、
比較で自分の痛いところが反応しているからですね。

明るく自然に輪に入っている人たちを見ると、
前に失敗したり傷ついたりした感じまで
一緒にぶり返してきます。

またああなるのが怖くて、
自分に蓋をしてしまう。

そうやって、
本来の自分らしさを出す前に、
先回りして閉じてしまうのです。


::me
それは痛感しています。

その場で明るく自然に話している人たちを見ると、
自分だけ入れない気がして、
縮こまってしまうことがよくあるんです。

不思議なのですが、
そうならない時もあって、
そんな時はすごく気分よく、
周りにとけこんで楽しく過ごせます。

なんでできる時とできない時があるんでしょう？

::

それは[余白](/articles/what-is-yohaku)のあるなしの違いですね。

自分の中が余白で満たされている時は、
多少の反応があっても
はねのけられるくらいの強さが保てます。

でも、反応に引きずられてそれに乗っかってしまった場合、
余白を失って、自分も見失ってしまいます。

そうなると、
バッドエンドルートに入ってしまうのです。

そこに気づかないと、
同じような場に行くたびに、
何度も同じ悪い選択をし続けて、
延々とループすることになります。


::me
嫌だな。

どうすればそのループに至らずに済む？

::

日頃から、余白の体幹を鍛えておくことです。
特に、何事もない平和でウキウキする時の
自分の余白を感じて覚えておくことが大事です。

一瞬一瞬、
いまは余白あるな、
ちょっと反応したぞ、
でもまだ余白はあるな、
みたいに、
日々の変化に敏感になっておくのです。


そうすると、
いざ、自分がいつも乱れやすい場があらわれても、
「きたな、大きいのが」
と気づくことができる。

無防備のまま食らうより、
ずっと柔軟に戻りやすくなります。

なにより今回の場面で大事なのは、
**その場で相手はまだ何も悪いことをしていない**
ということです。

まずは、
場そのものより先に、
**自分の中の反応が勝手に起きているのだ**と
気づくことです。

::me
勝手に起きた反応を
日頃から、余白の体幹を鍛えておくことで
素早く察知できたとして、
バッドエンドループから逃れることはできるの？
::

まだ慣れていないうちや、
反応が強い時、
気持ちが癒し切れていない時は、
そのループに向かう方が
楽だと思ってしまうことがあります。

だから、
必ず逃れられるとは限りません。

でも、
徐々に回避できることが増えていって、
痛みも少しずつ薄くなっていくのが
感じられるようになります。

ぜひ、違うエンディングを選んで、
あたらしい、あなたらしい輪に入ってみてくださいね。




::point
* 輪に入れないのは、自分の痛いところが反応しているから
* 日頃から余白の体幹を鍛えて、いざという時に備える
* その場で相手はまだ何も悪いことをしていない
* 自分の中の反応が勝手に起きているのだ
::
