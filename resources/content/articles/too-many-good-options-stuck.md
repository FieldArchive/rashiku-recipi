# どの選択にも良さが見えて、動けない時のレシピ

::me
やるべきことに、やる気が出ないんです。

でもこれ、
やるべきことが本当にやるべきことなのか？
って気づいてしまったことによる悩みっぽい。

::

なるほど。
その可能性は大いにありますね。

そこまで気づいているということは、
かなり余白の体幹が整っている証拠です。

それまでだったら
「お金が損したらもったいないから行く」
「どこかにいかないと損した気分になるから行く」
「食べておかないと損だから食べる」
のように、損得で物事を決める癖が完全についていて、
ある意味決めるのが楽だったはずです。

でも、
余白の体幹が備わってきて、
それらがあなたの中のざっちーの声だと気づき、
どんどん退治していったおかげで
ざっちーが無理に行かせよう、
やらせようとしていること結構あるって
わかってきたんだと思います。

::me
そう、その通り。

で、本当に私、それがしたいの？
という気持ちが出てきて、
今度は行動を決められなくなってしまったんです。

::

損得で決めないところまで来たから、
次は本音で選ぶ段階に来ています。

ざっちーで選ばなくなったぶん、
自由の悩みが出てきていますが、
それ自体はとてもよい傾向です。

自分らしさを育てる土台がようやく整ったからです。

ここからは、
「じゃあ、わたしは一体何がしたいのか？」を育てる段階に入ります。

いままで損得や他人の反応ばかり気にして
自分らしさを育てる余裕がなかったのではないですか？

急に何してもいいと言われても、
いったいどうしたらいいの？という気分かもしれませんね。

::me
うん。

急に自由になんでもしていいよって言われたようで、
戸惑っています。

だってね、
行かなきゃいけないことに行っても良さそうだし、
家にいるって選択しても良さそうで、
どの選択でも間違いなさそうな気がしてくるから困ってしまう。

::


確かにそうなりますよね。

前は、
損得や他人の判断みたいな、
わかりやすい基準がありました。

でもそれは、
あなたの本音のルートではなかったんです。

むしろ、
その基準に従うほど、
バッドエンドルートに入りやすいこともあった。

いまのあなたは
余白に気づくことによって
そのルートを選ばないようになった代わりに、
じゃあどのルートにいけばいいのか？と道に迷っている状態ですね。

もう外側の判断による選択ではなく、
自分の望みで選ぶ段階に来ているのです。

::me
どうやって道を決めたらいい？
どの道も正解っぽくて決められない
::

今のあなたは、
間違いを恐れて止まっているというより、
**外側の基準が外れたあと、内側の望みがまだはっきり育ちきっていない**
ところにいるんだと思います。

つまり、
動けない理由は不安ではなくて、
**自分の望みを読む感覚に、まだ慣れていない**

そうではないですか？


::me
まさにそれ。
不安はないんだけど、
決められない。

前より優柔不断になっちゃってない？
私。
::

今までは「私はなにがしたい？」という問いで選んでこなかったから、
すぐには答えが出ない状態ですね。

今は、
**望みで選ぶ前の、束縛がほどけた直後**
なんです。

幼い頃は、もっと素直に
「これがしたい」が出ていたこともあったはずです。 
でも長い時間をかけて、親や大人や規則の中で、
外側の基準で選ぶことに慣れてきたのではないですか。

そんな長期間培ってきたざっちーを退治し終わって、
じゃああなたはいまなにがしたい？
って言われても急には無理なのは仕方ありません。

優柔不断になってしまったと焦る必要はありません。

自分らしさを育てればちゃんとやりたいことも育っていきます。


::me
どうやって育てるの？
::


あなたらしさを育てるコツとしては、
人生で何がしたい？みたいな壮大なことではなく、
一瞬一瞬で余白を感じたのと同じように、
一瞬一瞬で自分がなにをしたいかを選ぶのです。

これは余白の体幹を鍛えるのと、そのうち同じ意味になっていきます。

自分らしいことをしていると、余白で満たされるというわけです。

損得を抜いて、ではどっちにいったらいいかわからない時はどうするか。

どっちでもいい。流れに任せる。
このあとどうなろうとそれが正解。
と思うのが余白ルートになります。

例えばいまのままだらだらしていてもいいし、
寝るのもいいし、
思いきって全然違う所に出かけるのもいい。
もちろん、行かなきゃと思っていた場所に行く気になってもいい。
要はなんでもいいのです。


::me
なんでもいいなんて答えになってないんですけれど
::

そうですね。

注意点はあります。

やらなかったこと、やれなかったことを悔やむのはいけません。
余白の体幹が身についたいまならあまりそう思うことはないかもしれませんが、
その発想はしないようにして、
少しでも自分がひらく方を選んで、
心から楽しめればいいのです。

そうやって
損得や他人の判断にしばられた選択ではなく、
自分らしい選択を日々とることによって、
他の誰でもない、
あなた独自の丁寧な毎日を過ごすことができるようになります。



::point

* 自分の望みを読む感覚に、まだ慣れていないだけ
* 損得や他人の判断にしばられた選択はもうしない
* 少しでも自分がひらく方を選び、やることなすこと楽しむ
::
