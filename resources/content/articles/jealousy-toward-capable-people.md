# 相手に引け目や嫉妬を感じて、気持ちがねじれる時のレシピ

::me
その人が嫌いってほどではないんだけど、
苦手というか、
その人に対するとき、
いつも妙にモヤモヤするのよ。

これ、なんだろう。
::

そういうモヤモヤが現れる時は、
大きく二つに分かれます。

あなたは、
まだ会ったことがない、それほど話したこともないのにモヤモヤしていますか？

それとも、
実際に会って話したあと、相手の態度や空気にモヤモヤしているのですか？

::me
今回は前者だな。

相手の経歴とか立場とかのせいかなぁ、
なんかえもいわれぬ嫌な感じになる。
::

その場合は、
まずこのレシピを試してみてください。

ただし後者でも、
相手は悪い人ではなさそうなのに、
なぜかモヤモヤする、
という場合はこのまま読み進めてください。

実際に無遠慮だったり、
踏みにじられた感じが残っていて不愉快になったのなら、
それは別の記事のほうが近いです。

→ [気分よかったのに、無遠慮に踏みにじられた時のレシピ](/articles/rude-attitude-stuck)



::me
そうね、
悪い人じゃなさそうってわかっているのに
その人と会うとなぜか
モヤモヤする時の気持ちに似ているかも。

気後れしちゃうというか、
なんだかもう負けたとか、
かなわなさそうとか、
そういう感じになってしまうのよ。
::


なるほど。

会う前からもう、その人とのあいだに勝敗を作ってしまっているんですね。

相手はまだ何もしていないのに、
こちらだけがもう
「見たくない」
「くやしい」
みたいな感じになってしまう。

それって、本当は
**相手を見ているようでいて、**
**実際には自分の見たくない内面を見ている**
のが原因なんですよ。

つらい現実を突き付けられるから
一瞬でもその人のことは考えたくない、いやだ、逃げたいとなってしまう。

::me
え？
なにそれ。

相手を見ているようで自分を見ている？
だって相手はそこにいるじゃない。
::

それがこのレシピの大きなポイントです。

会ってもいない、
会っていたとしても、わるい人じゃなさそうなのに
こちらが勝手に引け目を感じたり、モヤモヤしている場合、
それは**相手を見ているようで見てない状態**に突入しています。

それを「嫉妬」と呼ぶこともありますが、
その単語だと状態が正確にわかりにくいですよね。

その状態は、わかりやすく言うと、
**自分の未回収感が自分の中で勝手に起動している**
状態なのです。

::me
なるほどね。

私、相手のことを見ているつもりだったけど、
相手をきっかけにして
自分の痛いところを見ていたのか。

相手ではなく、自分を見ている。
それ、相手にも結構失礼な行為だね。
::

そうなんです。

だから
その人に会ったり考えたりしたときに
気分がモヤっとしたら、
まずやるべきは
**「私はいま、自分の痛いところに反応しているんだな」**
と気づくことです。

ここで質問ですが、
そういう痛い想いがあるって気づいた状態で、
相手はどう見えますか？

もう一つの似たようなモヤモヤ現象ででてきた、
相手が実際に無遠慮だったり、
踏みにじられた感じが残っていて不愉快になった場合と比べてどうですか？


::me
無遠慮で不愉快になったモヤモヤをもたらした人は、
はっきり私にとって不快な相手、
嫌いな相手だって言いきれるね。

でも、自分の内面の痛みからきたモヤモヤをもたらした人は、
どちらかというと、好きなのかもしれないというか、
もしかしたら気が合いそう、仲良くなれそうな気すらしてくる。

不思議だね。
おなじモヤモヤなのに、真逆だね。
::

相手が無遠慮な場合は、
「ああ、これは[ざっちー案件](/articles/what-is-zatchy) だね」
でわりと整理がつきます。

でも、そうでない場合は、
あなたの中の深いところが反応して
[余白](/articles/what-is-yohaku)が飛んでいる状態です。

ここをそのままにしていると、
本当だったら信頼できたかもしれない相手や、
本来なら入れたかもしれない居場所を、
自分で避け続けることになってしまいます。

**前者の場合は相手側に、後者の場合は自分側に、**
**余白が飛ぶきっかけがあった**ということになります。


::me
え。
じゃあ、後者の場合は、
相手のせいじゃないじゃない。

それって、相手がかわいそうじゃない？
::

わかってきましたね。

そうなんです。
今回の問題は、
**自分の過去の引っ掛かりが再浮上して起きている**ことなのです。

これは **トラウマ**という言い方もされていますね。


::me
嫉妬とトラウマって同じところからきているってこと？
::


同じではありません。
深さ、階層が違います。

嫉妬は表に出てきているモヤモヤで、
トラウマはそのもっと奥にある引っかかりです。

自分ができていないことができている、
自分が見たくなかったものを持っている、
そんな相手を前にした時に、
奥にあった引っかかりが刺激されて、
その結果として表にモヤモヤが出てくるのです。


::me
なるほど。

では奥底にあるトラウマを解消すれば、引っ掛かりが消えて、
表面にでてきた嫉妬のようなモヤモヤは消える、ということかな。
::

いいところまできましたね。

方向としてはそうです。

ただ、
そういう深い引っかかりは
すぐには消えません。

似たような相手や状況が来るたびに、
また反応してしまうことがあります。

そのたびに逃げたり、自分を卑下したり、
相手を悪く見たりする。

そういう状態は、
毎回バッドエンドを選ぶループに陥っているといえます。

::me
うわー、
それは嫌だな。
確かに毎回そうなってる。

これ、なんとかならないの？
::


バッドエンドループに気づく必要があります。
それも、ループに突入する前の入口で気づくことが大事です。

ループからはずれる方法を考えましょう。

まず最初に大事なことは、
この思いは相手のせいではないということに気づくことです。
その場のせいでもないですよね。
そして自分のせいでもない。

反応した瞬間にこれらのことに気づけると
ループの入口で気づけたことになります。

::me
相手のせいじゃないっていうのはわかる。

その場のせいでもないね。
別に緊張する場でも嫌な場でもなかった。

でも、自分のせいじゃない？
そんな風に考えちゃっているんだから。
::

自分が悪いと思っているうちは、まだバッドエンドルート上です。

あなたは最初からずっとモヤモヤしていたわけではないですよね。

その人の経歴や人となりを知った瞬間に、
自分の中で何かが反応して、
モヤモヤが発動した。

つまり、
誰かが悪いというより、
その瞬間にざっちー的な反応が割り込んできた
と考えるほうが近いです。


::me
え。
まさかこれも ざっちー案件！？
::

はい。
気づきましたね。

ただし今回は、
**相手からざっちーされたというより、**
**あなたの中でざっちー的な反応が起きていた、**
といえます。

相手そのものを見る前に、
過去の引っかかりや未回収感が勝手に起動して、
その反応で相手を見てしまっていたのです。


::me
そっか。

相手からぶつけられるざっちーも嫌だけど、
自分の中で起きるざっちーも、かなり嫌だね。
::


かなり嫌なのは確かですが、
**実はこの内面ざっちー、**
**気づくたびに減らしていける**んです。

相手から来るざっちーは、
こちらがどれだけ整っていても、
外から飛んでくることがあります。

でも、
自分の中で起きるざっちーは、
気づくたびに薄くしていくことができます。

::me
それはいいね。

なくしたい。
どうやればいいの？
::

大事なのは、
なくそうと力むことではありません。

その現象が起きた時に
「あ、またこれか」「あ、これは私側のざっちーだな」
と気づくこと、それだけです。

**モヤモヤの正体を自分のせいだと思わず、**
**ざっちーという変な生き物のせいだ**と思うようにするのがポイントです。

それなら、
退治しやすいですよね。

自分の中から湧き出る嫌な繰り返しを、
本音や現実そのものだと思わないことです。

ざっちー退治を繰り返すほど、
相手を雑に見てしまう時間は短くなっていきます。


::me
本来だったら素敵な関係になれるかもしれない人に対して
私から雑に接しちゃうなんて、嫌だ。

内面ざっちーがいなくなっていくと
どんないいことがあるの？
::

**本来なら出会えたかもしれない人、**
**話せたかもしれない人達に、**
**きちんと向き合えるようになります。**

**いままで避けていた世界も、**
**ひらいていきます。**

そして何より、
ざっちー的な反応に振り回されずに
**本来の自分らしさを取り戻す**
ことができます。

ぜひ、
内面のざっちーに気づいて、
反応を薄くしていって、
本来の自分らしさを思い出してくださいね。


::point
* 相手を見ているようで見ていないと気づく
* 自分の痛いところが反応していると気づく
* 一見本音っぽい心のざわめきを信じない
* 「内面のざっちー案件」だと気づく
* 「救いのあるざっちー案件」なので前向きに対処していこう
::
