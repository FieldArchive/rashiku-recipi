# やりたくないことから逃げたくて、別のことに没頭してしまう時のレシピ

::me
やるべきことや向き合わなきゃいけないことがあるのに、
つい、別のことに没頭してしまうんだよね。

これって現実逃避だよね。
良くないよね。
::

そう思ってしまうのも無理はありません。

普通に考えれば
「逃げている」
ように見えますよね。

まず大事なのは、自分をそんな風に責めないことです。

なぜ別のことに没頭したくなってしまうのか、
それが本当にいけないことなのか
対話してひも解いていきましょう。

ここで質問です。

それらをしている時、
あなたは自分から遠ざかっていますか？

それとも、
少し自分に戻っている感じがありますか？

::me
片付けしているときとか
趣味に没頭している時は、
自分に戻っている感じがあるかも。

でも、
お酒を飲んだり
ネットニュースやテレビをぼーっと見ている時は
自分に戻っている感じはないね。
むしろ遠ざかっている感じがする。

同じ逃げでも
感じがちがうね。
これってなんの違いがあるの？

::

大きな違いがあります。

ここで大事なのは、
「現実逃避」だとあなたが思っているものには
いくつか種類がある、ということです。

片付けや趣味に没頭している時は、
一見「逃げている」ように見えても、
**自分に戻れている**。

でも、
お酒やネットニュースやテレビをぼんやり見続ける時は、
その場ではラクになっても、
むしろ**自分から遠ざかっている**。

つまり、
同じように「やるべきことから離れている」ように見えても、
片方は**自己回復のための行動**で、
もう片方は**自分を見失う行動**なのです。

まず、この違いに気づきましょう。

::me
自分を見失う逃げは嫌だね。
どうせやるなら自分を取り戻すための逃げがいい。

でもなんで自分を見失うような逃げ方を選んでしまうんだろう。
::

現実も自分も何もかも忘れてしまいたい。
ただただ楽なほうを選んでしまうのは、
今の苦しさを感じないことのほうが最優先になるくらい、
[余白](/articles/what-is-yohaku)がなくなっているからです。

あなたの意志が弱いからではないんです。
**戻る力すら残っていないほど消耗している**
と見たほうが近いです。

麻痺するほうをあえて選ぶのは、
そこまで追い込まれているということなのです。

余白がないと、
あなたが何をしたい人なのかも、
何が好きなのかも、何が嫌なのかも、
わからなくなってしまうんです。

自分らしさがわからなくても、
少しだけ張りがゆるむものなら見つけられるかもしれない。
まずはそこからでいいんです。

::me
私はなにをしたいのか、
なににワクワクするのかなんて
ずっと考えてこなかったから
いきなり自分のことなんて考えられないよ。

逃げるほうが楽。
::

生きることが
感じることではなく
耐えることになっている時は
逃げるほうが楽と感じてしまいがちです。

でも、だからといって
慣れ親しんだ楽なルート、
自分を飛ばして安心できるルートばかり選んでいると、
余白に戻る感覚は
なかなか育っていきません。

余白があると、少し自分に戻れる。
少し自分に戻れると、
生きるのが少し楽になると思ってください。

だからまずは、
バッドエンドルートの入り口に気づいて、
なるべく入らない道を選んでいくことが大事なのです。

::me
どうやったら入り口で気づけるの？
どうやって入らない道を選べるの？
::

自分にとって余白のある状態はなにかを
普段から気づくようにしておくことです。

行き詰ってからいきなり余白に気づこうとしても
もう楽な道、この道しかないと思い込みがちです。

そうではなく、普段から自分が何に幸せを感じるか、
何が心地よいか、ゆるんだ感じがするか、
余白は人によってまったく違います。

余白を見つけるヒントは、
大きく二つあります。

まずは寝ること。
「なにもせず、余計なことを考えず、身体を休めること」です。
これは自分の余白を取り戻すのにかなり効果的です。

もう一つは
「やるべきことや向き合わなきゃいけないことがあるのに、
つい、別のことに没頭してしまうことを心置きなくやる」です。

::me
え？

それって現実逃避しているって
私が思っているものたちじゃない。
::

そうなんです。

さきほど言った通り、
現実逃避には二種類あります。

自分を見失うものと、自分を取り戻すもの。

やりたくないことをやらなくてはいけなくて、
でも違うことをやりたくなった時は
「自分の余白回復法を見つけるチャンス」だと思ってください。

一つはバッドエンドルートに直結する逃げですが、
もう一つは寝ることと同じくらい、自分の余白を取り戻すのに効果的な行動なのです。

そこで提案です。

やらなきゃいけないこととともに、
１枚の紙を用意して、
「現実逃避」したくなったとき、
それをする前に必ずその行動を紙に書いてみてください。

１日できれば３日、それを続けてみると
自分の逃げの傾向、逃げ癖が見えてくると思います。

::me
私はパソコン作業の仕事が多くて、
気づくと
・ネットニュースチェック
・SNSチェック
・Google検索
ばっかりやってたな。
::

ここで質問です。
それらは自己回復のための行動と、自分を見失う行動、
どちらに属すると思いますか？

::me
ニュースやSNSを見るのは、
自分を見失う行動、だったよね？
::

行動の名前だけでは決まりません。

同じニュースやSNSでも、
自分の興味や知りたいことに沿って見ているのか、
ただ現実から逃げたくてぼんやり流されて見ているのかで、
意味がまったく変わってきます。

::me
自分に興味があることを
だらだら見ていたよ。

気づいたらあっという間に時間がとけてた。
::

それなら、
今のあなたの場合は
自己回復のほうに寄っています。

なぜなら、
その時のあなたは
何も感じないように麻痺していたのではなく、
自分が気になるものを
自分で追っていたからです。

そうではなく、
現実から逃げたいから
何も考えずに延々とニュースをむさぼり見る、
同じツライ境遇の人のSNSで自分を慰める、
テレビをただぼーっと見ている。

これらとの違い、わかりますか？

::me
わかる。

確かに、それこそ自分を見失っている行動だね。
::

そうです。

いまやっていることから逃げたいと思った時に、
どっち方向に逃げることを選ぶか。
そこが大きな分かれ道になります。

頑張ってみても、どうしても
自分を壊す方を選んでしまう場合は、
一度、[なぜかバッドエンドになってしまう時のレシピ](/articles/trying-hard-but-bad-ending) を読んでみてください。

これは「無遠慮な人に対してどうしても反応してしまう場合に効果的なレシピ」ではありますが、
人ではなく、自分に読み替えて読んでみてください。
自分に無遠慮な場合にも使えるレシピです。

これを読んでもわかるように、余白をつくることが大きなポイントです。

::me
余白を取り戻す行動を良しとしてしまうなら、
もう、延々とそれをやり続けちゃって、
止まらなくて、
本来やらなきゃいけないことを
すっぽかしちゃっていいってことになるよ。
::

そこは線引きが必要です。

そこはきちんと線引きする必要がありますね。
気をつけないと、それらの中に、バッドエンドルートがひそんでいます。

では、ここで二つ大事な質問をします。

**本当にそれしたいですか？**

**それをしたことで自分を見失わないですか？**

快楽にハマって自分を見失うような行為は起こしてはいけません。
ビールや甘いものも適量ならいいですが、
現実から逃げるために自分を見失うくらいの量を摂取するのはやりすぎです。
線引きが大事です。

自分を見失うような「本当の現実逃避」と、
疲れて自分を見失ってしまいそうな現実から「逃避」することとは、
似ているようで全く違うのです。

ただ、時間を忘れてビールのことを調べたくなる、
飲み比べが大好き、
居酒屋巡り大好き、
甘いものも食べ歩くの大好き、
とまでいくと、
悪いこと、現実逃避ではなく、
自分らしさを生み出すなにかになっていきます。

それはおさえてはだめです。
それでいいんだと自覚して、卑屈にならず、罪悪感ももたずに、もっと楽しく、心置きなく、やるべきです。

ここまでをまとめますね。

やる気にならない時は、
いきなり「やるべきこと」に向かうのではなく、
まず余白を取り戻すことが先かもしれません。

あなたは、
余白を求めているのかもしれないし、
余白がなくなるほどすり減っているのかもしれない。

そんな時は、
「あー、そんなことやっている場合じゃないのに」
というざっちーの声には耳を貸さず、
先に気持ちを整えて、
余白を取り戻すことを考えてください。

余白を生み出すことなら遠慮せずやってみる。
余白を生み出さないことなら、よほど疲れているので何も考えずに寝る。

そんな状態のまま
やらなきゃいけないことに向かっても、
雑になったり、
かえって長引いたりしやすいのです。

ざっちーをまずは退治して、
気持ちが整った後、心行くまで余白を取り戻す、
という順番でいくと、
結果的にやらなくてはいけないことを効率よく、
しかも良い形で仕上げることができます。

::point
* 現実逃避には二種類あると気づく
* 自分を取り戻す逃げと、自分を見失う逃げを分ける
* やる気が出ない時は、まず余白回復が先かもしれない
* 「そんなことしてる場合じゃない」はざっちーの声かもしれない
* 先に整えてから向かったほうが、やるべきこともうまくいきやすい
::
