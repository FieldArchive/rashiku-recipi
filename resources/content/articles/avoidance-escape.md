# (仮) やりたくないことから逃げたくて、別のことに没頭してしまう時のレシピ

::me
ここは仮置きです。

やるべきことや向き合いたくないことがあるのに、
別のことに没頭してしまう時のレシピを、
あとでここに移します。
::



- 締め切りや問題から目をそらして、理由をつけて先延ばしにしてしまう
- 目の前にあるやるべきことや、直面している困難から、意図的に意識をそらして避けてしまう
- 本来すべきことじゃなくて、趣味とか娯楽とか、楽しいことや楽なことに没頭してしまう


::me
頭ではわかるんだけどね、
でもやっぱりこんなことやっている場合か？
って思っちゃうんだよ。

ちょうど**試験期間中で勉強しなきゃとわかっているのに**
**机の片付けや部屋の模様替えに没頭しちゃう**みたいな感じ。

現実から逃げているんだよなって
やりながら自己嫌悪におちいって凹んじゃう。
::

なるほど。
では一つ質問します。

やらなきゃいけないこと以外のことに夢中になっている時、
趣味や調べもの、創作や片づけなど、
そのやっていること自体は楽しいですか？


::me
うん、もちろん楽しいよ。

普段はやろうとも思わないんだけど、
なぜかすごくやりたくなるんだよ。

やっぱりこれって現実逃避なんじゃない？
::

まだ考え方がバッドエンドルート上にいますね。

ではちょっと質問の趣旨を変えてみます。

急に無遠慮に気持ちを踏みにじられた時、
あなたはどうやってその気持ちをご機嫌にもどしますか？

::me
それは解決できるよ。

店員や知人から、
悪気なく急に嫌なことをされて
一気に気落ちした場合でしょ、

誰も何も悪くなくて「その瞬間だけが悪かった」
失礼なことをされても気にしないルートを選べ、だったよね。

::

そうです。

[気分よかったのに、無遠慮に踏みにじられた時のレシピ](/articles/rude-attitude-stuck)で話しましたが、
一見、「失礼な態度をとるその人自身が悪」という考えがもっともらしくて正解っぽいわけですが、
そのルートを辿り続けても、いつまでたっても[余白ルート](/articles/what-is-yohaku)に向かいません。

元々悪かったわけでもないもの達のせいにするのでは解決しないのです。
同じ現象が起きたら何度でも同じ残念な思いにかられてしまいます。

それと同じことが今回もおきています。

::me
ん？
失礼な態度をとる第三者なんてどこにもいないけれど？

今回は自分との闘いって感じなんだけど。
::

そう見えますが、実は構造は同じです。

人が出来事に変わっただけです。

ではあらためて
３つ質問をします。

１つ目
「不安な気持ちになったのはその場のせいだとは思いますか？」

::me
いや、その場のせいじゃないと思う。

だって、やっている最中は夢中だったり、
楽しかったり、
ちゃんと通っている感じがあった。
::

２つ目
「あなたの気持ちはなぜご機嫌な状態からはずれたのですか？」

::me
これって一銭にもならないよねって思った瞬間に、
急に現実が怖くなって、ご機嫌ではいられなくなった。

そう思う前まではやりたいことをやれて、むしろすごい楽しかった。
::

３つ目
「もともとその出来事はずっと不安な出来事でしたか？」

::me
いや、ちがうね。

やっていて、ずっと楽しかった。
楽しすぎて、ふと我に返って、
あれ、これって良くないんじゃない？
って手がとまって、そこから先急に不安にかられて何もできなくなった。
::


解決に向けての答えが出てきましたね。

つまり、
場はもともと悪くなかった、
あなたの機嫌ももともと悪くなかった、
出来事ももともと悪い出来事だったわけではない、
ということです。

今回の出来事に対しても
「雑な現象が一瞬起きたせい」と考えるのが解決に向けての一番のポイントなのです。

::me
そんな風にいわれても、
現実には実際にお金にならないことをずっとやっているし、
ちゃんと不安に思うべきなのでは？

何も悪くないって、楽しみ続けるなんて、
やっぱり単なる現実逃避じゃないの
::

人が出来事に変わった分、誰のせいにもしにくいので、自分を悪にしがちですね。

では「現実逃避」とは一体なんでしょうか？

::me
締め切りや問題から目をそらして、
理由をつけて先延ばしにすること。

本来すべきことじゃなくて、
趣味とか娯楽とか、
楽しいことや楽なことに没頭してしまうこと。

目の前にあるやるべきことや、
直面している困難から、
意図的に意識をそらして避けようとすること、だよね。
::

そうです。

それらのことを意識して改善しようと思うこと自体は何の問題もありません。

問題は、
その状態を引き起こしている自分を悪だと思うこと、
その出来事自体を悪いことだと思ってしまうこと、なのです。

無遠慮に他人に気持ちを踏みにじられた時と同じく、
もともとどれも悪くなくて、
やっていたことそのものが悪かったわけでもありません。

ただその途中で、
**一瞬、雑に不安な気持ちが割り込んできた。**
それだけなんです。

ちょっとわかりにくいですが、
これも、[ざっちー案件](/articles/what-is-zatchy) なんですよ。

::me
ざっちー案件だと思うようにして、
瞬間の雑な想いを飛ばして余白に戻れたとしても、

本来すべきことじゃない、
趣味とか娯楽とか、
楽しいことや楽なことに没頭してしまうことは良くないことなのでは？
::

不安な気持ちを「雑な一瞬の想い」として切り分けられたことはとても良いです。

ただ、
ざっちー案件だと気づいて一瞬の不安からは戻れたとしても、
「でも、そもそもこんな楽しいことに没頭していていいの？」
という疑いが残ると、
またすぐ不安が戻ってきてしまいますよね。

だから今回はもう一段先で、
「本来すべきことではないことに惹かれるのは、本当に悪いことなのか？」
を見直す必要があります。

「楽しいことをやっていていいの？それは逃げじゃないの？」

その答えをズバリ言うと
「**はい。楽しいことを最優先してやるべきです。逃げではないです**」
となります。

::me
えええ？？

逃げじゃないの？
どう考えても逃げでしょう。
::



元々悪かったわけでもないもの達のせいにするのでは解決しないのです。
同じ現象が起きたら何度でも同じ残念な思いにかられてしまいます。


::point
