# やりたくないことから逃げたくて、別のことに没頭してしまう時のレシピ

::me
やるべきことや向き合わなきゃいけないことがあるのに、
つい、別のことに没頭してしまうんだよね。

これって現実逃避だよね。
良くないよね。
::

そう思ってしまうのも無理はありません。

普通に考えれば
「逃げている」
ように見えますよね。

まず大事なのは、自分をそんな風に責めないことです。

なぜ別のことに没頭したくなってしまうのか、
それが本当にいけないことなのか
対話してひも解いていきましょう。

ここで質問です。

それらをしている時、
あなたは自分から遠ざかっていますか？

それとも、
少し自分に戻っている感じがありますか？

::me
片付けしているときとか
趣味に没頭している時は、
自分に戻っている感じがあるかも。

でも、
お酒を飲んだり
ネットニュースやテレビをぼーっと見ている時は
自分に戻っている感じはないね。
むしろ遠ざかっている感じがする。

同じ逃げでも
感じがちがうね。
これってなんの違いがあるの？

::

大きな違いがあります。

ここで大事なのは、
「現実逃避」だとあなたが思っているものには
二種類ある、ということです。

片付けや趣味に没頭している時は、
一見「逃げている」ように見えても、
**自分に戻れている**。

でも、
お酒やネットニュースやテレビをぼんやり見続ける時は、
その場ではラクになっても、
むしろ**自分から遠ざかっている**。

つまり、
同じように「やるべきことから離れている」ように見えても、
片方は**自己回復のための行動**で、
もう片方は**自分を見失う行動**なのです。

まず、この違いに気づきましょう。

::me
自分を見失う逃げは嫌だね。
どうせやるなら自分を取り戻すための逃げがいい。

でもなんで自分を見失うような逃げ方を選んでしまうんだろう。
::

現実も自分も何もかも忘れてしまいたい。
ただただ楽なほうを選んでしまうのは、
今の苦しさを感じないことのほうが最優先になるくらい、
[余白](/articles/what-is-yohaku)がなくなっているからです。

あなたの意志が弱いからではないんです。
**戻る力すら残っていないほど消耗している**
と見たほうが近いです。

麻痺するほうをあえて選ぶのは、
そこまで追い込まれているということなのです。

余白がないと、
あなたが何をしたい人なのかも、
何が好きなのかも、何が嫌なのかも、
わからなくなってしまうんです。

自分らしさがわからなくても、
少しだけ張りがゆるむものなら見つけられるかもしれない。
まずはそこからでいいんです。

::me
私はなにをしたいのか、
なににワクワクするのかなんて
ずっと考えてこなかったから
いきなり自分のことなんて考えられないよ。

逃げるほうが楽。
::

生きることが
感じることではなく
耐えることになっている時は
逃げるほうが楽と感じてしまいがちです。

でも、だからといって
慣れ親しんだ楽なルート、
自分を飛ばして安心できるルートばかり選んでいると、
余白に戻る感覚は
なかなか育っていきません。

余白があると、少し自分に戻れる。
少し自分に戻れると、
生きるのが少し楽になると思ってください。

だからまずは、
バッドエンドルートの入り口に気づいて、
なるべく入らない道を選んでいくことが大事なのです。

::me
どうやったら入り口で気づけるの？
どうやって入らない道を選べるの？
::

自分にとって余白のある状態はなにかを
普段から気づくようにしておくことです。

行き詰ってからいきなり余白に気づこうとしても
もう楽な道、この道しかないと思い込みがちです。

そうではなく、普段から自分が何に幸せを感じるか、
何が心地よいか、ゆるんだ感じがするか、
余白は人によってまったく違います。

今回のような場合で余白を見出すには
大きく二つの方法があります。

まずは寝ること。
「なにもせず、余計なことを考えず、身体を休めること」です。
これは自分の余白を取り戻すのにかなり効果的です。

もう一つは
「没頭してしまうことを心置きなくやること」です。

::me
え？

それって現実逃避しているって
私が思っているものたちじゃない。
::

そうなんです。

さきほど言った通り、
現実逃避には二種類あります。

自分を見失うものと、自分を取り戻すもの。

やりたくないことをやらなくてはいけなくて、
でも違うことをやりたくなった時は
「自分の余白回復法を見つけるチャンス」だと思ってください。

一つはバッドエンドルートに直結する逃げですが、
もう一つは寝ることと同じくらい、自分の余白を取り戻すのに効果的な行動なのです。

そこで提案です。

やらなきゃいけないこととともに、
１枚の紙を用意して、
「現実逃避」したくなったとき、
それをする前に必ずその行動を紙に書いてみてください。

１日できれば３日、それを続けてみると
自分の逃げの傾向、逃げ癖が見えてくると思います。

::me
私はパソコン作業の仕事が多くて、
気づくと
・ネットニュースチェック
・SNSチェック
・Google検索
ばっかりやってるな。
::

それらは自己回復のための行動と、自分を見失う行動、
どちらに属すると思いますか？

::me
ニュースやSNSを見るのは、
自分を見失う行動、だったよね？
::

行動の名前だけでは決まりません。

同じニュースやSNSでも、
自分の興味や知りたいことに沿って見ているのか、
ただ現実から逃げたくてぼんやり流されて見ているのかで、
意味がまったく変わってきます。

::me
なるほど。

後者は、
人の考えや外から入ってくる情報に
引っぱられているだけって感じ。

行動に対して
自分が通っていないね。
::

そうです。

そこまで違いがみえてくると、
逃げたくなった時にどちらに逃げたらいいか、
もうわかりやすいですよね。

ここまでをまとめると、
やっていることから逃げたくなったら、
それが自己回復か外にひっぱられているかを感じとり、
自己回復（余白を取り戻す）ことに
全力でうちこむ、です。

::me
え、
それだと、
余白を取り戻すことを優先して、
本来やらなきゃいけないことを
すっぽかしちゃっていいってことになる

寝たり、趣味に没頭したりすべきってこと？
::

そうです。

やる気にならない時は、
無理して「やるべきこと」に向かうのではなく、
まず余白を取り戻すことが先、だと思うようにしてください。

余白を生み出すことなら遠慮せずやってみる。
余白を生み出さないことなら、よほど疲れているので何も考えずに寝る。

なぜなら、
そんな状態のまま
やらなきゃいけないことに向かっても、
雑になったり、
かえって長引いたりしやすいからです。

::me
やろうとしても、
そんなことやっている場合じゃないでしょって
頭の中で渦巻くんだけど
::

それはあなたの中に潜む [ざっちー案件](/articles/what-is-zatchy) です。

やらなきゃいけないことから逃げること
自分の好きなことを優先すること
を悪だと心で考えている場合に、
その場面の瞬間に出没する
謎の生き物だと思うとわかりやすいです。

バッドエンドルートに向かわせようとする
ざっちーの声には耳を貸さず、
余白を取り戻すことを最優先してください。

気にせず好きなことをやって
余白を取り戻す行動を続けていくと、
自然とざっちーの出没回数は減っていきます。

本来、
やらなきゃいけないことから逃げることは悪ではないし、
自分の好きなことを優先することも悪いことではないと悟ると退治が早いです。


::me
本当にしたい、
したことで自分を見失わないと思ったことは
何でも率先してやったほうがいいんだね？
::

はい。

それらはむしろ、
おさえこまないほうがいいです。

それでいいんだと自覚して、
卑屈にならず、
罪悪感を持ちすぎず、
自分を戻すための大事な動きとして
扱ってください。

そうすると、
「やらなきゃいけない」「やるべき」と否定気味に思っていることも
その必要性に応じて心を込めて「やりたい」と思うようになりますし、
さらに、自分の本当にやりたいことに気づいてその道に向かって進んでいけるようになります。

そのために
まずは「日々が余白で満たされること」です。

どの行動も気負わず、没頭しても引け目を感じることがなくなっていきます。


::point
* 現実逃避には自分を取り戻す逃げと、自分を見失う逃げがある
* やる気が出ない時は、まず余白回復が先
* 「そんなことしてる場合じゃない」はざっちーの声
* 先に整えてから向かったほうが、やるべきこともうまくいく
::
