# お金が足りなくなる気がして、買う決断ができない時のレシピ

::me
お金がすぐに足りなくなるわけじゃないんだけど、
大きな買い物をしようとすると、
手持ちがすごく減ってしまう気がして不安になる。

大丈夫って思おうとしても、
不安がとれないから買う決断ができない。
::

そのえもいわれぬ不安の原因を紐解いていきましょう。

いまのあなたは、
本当に今すぐ困る現実を見ているというより、
まだ起きていない未来の不足を
先回りして怖がっているんですね。

買うことによって本当にお金がなくなってしまう、
借金をするような買い物は止める必要があります。

でも今回のように、
買える力はあるのに
起きていない未来のせいで止めている場合、
本当は必要なものまで
気持ちよく受け取ることが
できなくなってしまいます。



::me
それは嫌だな。

どうしたらこの不安をなくせる？
::

まずこの不安は現実そのものではなく、
まだ起きていない未来への不安だと気づくことです。

わかりにくいかもしれませんが、
あなたの心の中に ざっちーという謎の生き物がいて、
その生き物が余計なことをあなたに吹き込んでいる、と考えてみてください。

不安の気持ちに対して、
あなたが考えていると思うと余白がなくなりますが、
あなた以外の何かが問いかけていると思うと
余白がうまれます。

そして、
「なんで不安なんだっけ？」
と問いかけてみてください。
そうすることで、漠然とした不安から、
現実味のある不安に変わります。

まずそれが大きな一歩です。


::me
なんで不安なのか？

だって、お金が大量になくなるじゃない。
不安でしょ。
::

それでは答えになっていませんよね。

もしかして、すぐ後に大金が入るあてがあるかもしれませんし、
円の大暴落があって、
あの時欲しいもの買っておいてよかったね、となるかもしれませんよね。

::me
えー、
そんな未来のことなんてどうなるかわからないじゃない
::

そうなんです。

そんな未来のことなんてわからないのに、
最初はその未来のせいで不安に陥ってたのです。

その矛盾にまず気づきましょう。

資金繰りが悪くなるかもしれないし、
すごく良くなるかもしれない。

それは、
「欲しいものを買う」
という行動とは完全に切り離して考えるのが大きなポイントです。

まったくお金に問題がなかったらその商品を買いますか？
欲しいものとは違うけれど割引で安くなっている商品と
思い通りのものだけれども高い商品。
どちらを買いますか？

::me
まったくお金に問題がなければ、
思い通りのものを買いたいね。
割引も気になるけれど。
というか、普段は妥協して割引商品ばっかり買っているよ。
::

よく言われる言葉に、
**「迷う理由が値段なら買え、**
**買う理由が値段ならやめておけ」**
というのがあります。

まさにその言葉をあなたにおすすめします。

一瞬一瞬、常に、
**物を買う時は、自分がお金に困っていない前提で選ぶ**
ようにしてみてください。

それが、あなたらしさ、あなたらしい生き方に直結します。

その行動が身についていれば、
もう答えが出ていますよね。

不安な気持ちはあなたの心の中に潜む ざっちーの戯言。
お金を基準に買うものを考えない。
となると、あなたの最初の問題はどうなりますか？

::me
いくらであろうと、
一番欲しいと思った商品を妥協せずに買う、かな？
::

そうです。

ただし、
身の丈を超えた無謀な買い物、
ブランド品や高級車などを買う、
となると、ちょっと話は変わってきますよ。

それはあなたの中の
「満たされてない」
「もっと見栄を張りたい」
「私はもっとすごい人なんだ」
みたいなざっちーの声を聞きすぎている可能性が高いです。

そうではなくて、
身の丈にあっていて、
自分のこの先の道を明るく照らしてくれる商品、
それを持つことで自分の世界が正しく開かれるような買い物は、
一考する価値があります。

不安に押しつぶされず、
自分の可能性や、
大きな買い物をする力があると信じて、
お金のせいで諦めてしまわないようにしてくださいね。



::point
* 未来の不安はざっちー案件
* 不安基準で選ばず、自分の本音基準で選ぶ
* 一瞬一瞬、お金で判断しない買い物術を会得する
* 良いものを手にする自分の可能性を信じる
* 身の丈を超えた買い物欲はざっちー案件
::
